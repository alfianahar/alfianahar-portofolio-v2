## Problem

Cosmos-based chains lack native price oracle mechanisms, making real-time token valuation and USD-denominated transaction limits impossible without external centralized price sources.

## Solution

Built a Cosmos SDK chain with a decentralized oracle module integrating Skip Connect and Pyth price feeds. The module supports multi-currency feeds and dynamic transfer/daily limits that adapt to real-time token price fluctuations.

## Outcome

A fully functional blockchain module with queryable price feeds, dynamic USD-based transfer limits, and daily limit enforcement, enabling on-chain applications to rely on current token valuations without trusting a single price source.
